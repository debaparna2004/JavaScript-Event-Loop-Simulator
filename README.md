# JavaScript Event Loop Simulator

An interactive visualization tool to understand how the JavaScript Event Loop works behind the scenes.

## Features

- Simulates:
  - Call Stack
  - Web APIs
  - Callback Queue
  - Event Loop
- Visual execution flow
- Task scheduling simulation
- Helps beginners understand asynchronous JavaScript
- Interactive UI built with HTML, CSS, and JavaScript

## Tech Stack

- HTML
- CSS
- JavaScript

## Concepts Demonstrated

- Synchronous vs Asynchronous execution
- setTimeout()
- Callback Queue
- Call Stack execution
- Event Loop behavior

## How to Run

1. Clone the repository

```bash
git clone https://github.com/debaparna2004/JavaScript-Event-Loop-Simulator.git
```

2. Open the project folder

3. Run `index.html` in your browser
